The Bitcoin payment solution

An open source, free and easily integrable Bitcoin payment gateway, designed for online shops.

Features

Compatible

BytePayr is easily integrable with any other payment solution that you're already using in your store.

Free

BytePayr has no fees, is completely free and open source. The amount the client payed is the amount you get.

Secure

With BytePayr, all the components of the gateway run in your server and you have total control over them.

Customizable

You can use one of our frontends for BytePayr or use directly the REST API and build it yourself.

How it works

Frontend

This is layer that interacts with the user and that is integrated in your website. You can use one of our gateway frontends or build your own using our API.

Gateway

Here is with the magic happens. This layer constitutes the core of BytePayr, offering a REST and Socket.io API and interacting with the Bitcoin wallet.

Wallet

The safe where the payments will be received and stored. We use the open source and well tested bitcoin client, so your money is completely safe.

BytePayr can be easily integrated in your store if you use PHP or Node.js, using one of our frontends. If your site is written in another language, you can use the standalone frontend, and if none of this works for you, we offer an extremely simple REST and Socket.io API.
